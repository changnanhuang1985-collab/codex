# Ariba-S/4 Integration POC

## 概要

本フォルダは、以下案件を題材とした `SAP BTP / Integration Suite` ベースの POC 一式です。

`SAP Ariba - SAP S/4HANA 連携高度化に向けた SAP BTP / Integration Suite 活用ケース`

本 POC パッケージは、以下を目的として作成しています。

- 対日案件を想定した実務レベルのプロジェクト成果物サンプル
- 次回案件で再利用可能なテンプレート集
- PM、要件定義、設計、開発、試験、受入までを通したレビュー可能な標準成果物

## フォルダ構成

- `00_Project_Management`
- `01_Project_Preparation`
- `02_Requirements`
- `03_Architecture`
- `04_Basic_Design`
- `05_Detail_Design`
- `06_Development`
- `07_Testing`
- `08_Acceptance`
- `09_Operations_Transition`
- `10_Templates_and_Samples`

## ファイル形式方針

- 表形式の成果物は `CSV` で作成し、Excel でそのまま利用できるようにする
- 説明文中心の成果物は `Markdown` で作成する
- 正式文書の記載言語は日本語を基本とし、技術オブジェクト名のみ必要に応じて英語を残す
