# Troubleshooting Guide

## 目的

本文档用于沉淀 `SAP BTP Integration` 常见问题及排查方法。

## 1. iFlow 执行失败

### 常见现象

- 消息处理失败
- `Exception Subprocess` 被触发
- `MPL` 中出现错误状态

### 排查建议

- 查看 `Message Monitoring`
- 打开 `Trace`
- 检查报文内容与字段映射
- 检查目标系统返回报错
- 检查脚本是否抛出异常

## 2. API 调用失败

### 常见现象

- `401 / 403`
- `404`
- `429`
- `500`

### 排查建议

- 检查 `OAuth` 或其他认证配置
- 检查 `API Proxy` 路径
- 检查限流配置
- 检查后端服务可用性

## 3. SFTP / 文件类接口失败

### 常见现象

- 文件未读取
- 文件重复处理
- 目录不可访问

### 排查建议

- 检查连接配置
- 检查用户权限
- 检查文件命名规则
- 检查轮询周期与归档策略

## 4. IDoc / SOAP / OData 接口失败

### 排查建议

- 核对协议配置
- 核对 `endpoint`
- 核对证书与认证方式
- 检查字段结构与命名空间

## 5. Event Mesh 相关问题

### 常见现象

- 事件未投递
- 队列无消息
- 消费者未收到事件

### 排查建议

- 检查 `Topic` 配置
- 检查 `Queue` 绑定关系
- 检查生产者与消费者配置
- 检查权限与连接信息
