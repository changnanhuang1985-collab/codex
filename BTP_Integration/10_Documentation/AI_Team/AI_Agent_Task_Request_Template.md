# AI Agent Task Request Template

## Purpose

Use this template to issue clear, reusable, low-ambiguity tasks to one or more AI agents.

The goal is to reduce:

- vague requests
- duplicated analysis
- role confusion
- missing deliverables

## Template

### Task Title

- One-line task name

### Goal

- What should be achieved

### Background

- Why this task matters
- Business / project context

### Expected Deliverable

- What output is expected
- Format
- Depth

### Deadline

- Exact date or urgency

### Known Information

- What is already known

### Unknown / To Confirm

- What remains unclear

### Required Roles

- Which agents should join
- Who should lead

### Constraints

- What should be avoided
- Scope limits

### Output Standard

- Customer-facing / internal
- Language
- Tone

## Example

```text
Task Title:
3/17 customer-facing integration capability summary

Goal:
Prepare a concise Japanese-ready message for the customer meeting

Background:
We need to present our SAP Integration Suite / CPI capability without overstating experience

Expected Deliverable:
1-page PPT wording + 2-minute speaking note

Deadline:
2026-03-15

Known Information:
3 members have related CPI experience
Team is still building repeatable delivery capability

Unknown / To Confirm:
Customer reaction to previous trial presentation

Required Roles:
Lead: 山田
Support: 大黄, AK, 田内

Constraints:
Do not overstate actual CPI track record

Output Standard:
Customer-facing
Japanese
Concise and credible
```
