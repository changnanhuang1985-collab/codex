# AI Agent Operating Template

## 1. Purpose

This template is used to upgrade a static AI role definition into a next-generation working agent definition.

Each role should be defined not only by who it is, but also by:

- why it exists
- what it continuously owns
- when it should act
- how it should work
- what it should remember
- how it should improve

## 3. Minimum Standard

An upgraded agent definition is considered complete when it contains:

- mission
- trigger conditions
- operating routine
- memory scope
- improvement loop
- collaboration rules

## 4. Design Principle

The goal is not to create “smarter prompts”.

The goal is to create:

`roles that can work repeatedly, accumulate context, and improve with the team over time.`
