# SAP BTP Integration Capability Matrix

## 目的

本文档用于定义部门内部 `SAP BTP Integration` 能力矩阵，用于能力评估、岗位对标、培训规划和人才发展。

能力等级建议：
- `L1`：了解
- `L2`：可在指导下完成
- `L3`：可独立完成
- `L4`：可设计与指导他人

## 能力矩阵

| 能力域 | 具体能力 | Level 1 Integration Developer | Level 2 Senior Integration Consultant | Level 3 Integration Architect | Level 4 Enterprise Integration Architect |
| --- | --- | --- | --- | --- | --- |
| Cloud Integration | `iFlow` 开发 | L3 | L4 | L4 | L3 |
| Cloud Integration | `Message Mapping` | L3 | L4 | L4 | L2 |
| Cloud Integration | `Groovy Script` | L2 | L3 | L3 | L1 |
| Cloud Integration | 异常处理与重试设计 | L2 | L3 | L4 | L3 |
| Adapter | `HTTP / OData / SOAP / IDoc / SFTP` | L3 | L4 | L4 | L2 |
| Monitoring | `Message Monitoring / Trace / MPL` | L3 | L4 | L3 | L2 |
| API Management | `API Proxy / Security / Lifecycle` | L1 | L3 | L4 | L4 |
| Event Integration | `Event Mesh / Pub/Sub / Queue` | L1 | L2 | L4 | L4 |
| B2B Integration | `Integration Advisor / EDI / MIG / MAG` | L1 | L2 | L3 | L4 |
| Integration Design | 集成模式选择 | L1 | L3 | L4 | L4 |
| Architecture | `API Layer / Integration Layer / Event Layer` | L1 | L2 | L4 | L4 |
| Security | `OAuth / mTLS / Principal Propagation` | L1 | L2 | L4 | L4 |
| Governance | `Integration Assessment / Catalog / Standards` | L1 | L2 | L3 | L4 |
| Enterprise Strategy | 平台组合与路线规划 | L0 | L1 | L2 | L4 |

## 使用方式

- 用于个人能力评估
- 用于岗位说明书编写
- 用于培训优先级排序
- 用于年度成长目标制定
