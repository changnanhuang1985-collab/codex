# SAP BTP Integration

## Integration Capability

### Integration Development

- Cloud Integration
  - integration flow
  - message transformation
- API Management
  - API gateway
  - lifecycle
- Event Mesh
  - event-driven integration
- Open Connectors
  - SaaS connectors

### B2B Integration

- Integration Advisor
  - EDI mapping
  - MIG / MAG
  - 自动 mapping 建议

### Integration Governance

- Integration Assessment
  - 推荐合适的集成模式
- Integration Catalog
  - 集成资产库
  - API / iFlow / pattern catalog
- Integration Landscape Visualization
  - 系统与集成关系可视化

## Integration Styles x Capability

### Process Integration

- Capability
  - CPI
- 目标
  - 跨系统业务流程
- 场景
  - Employee Onboarding

### Data Integration

- Capability
  - CPI
- 目标
  - 数据复制 / 同步
- 场景
  - Business Partner replication

### API-based Integration

- Capability
  - API + CPI
- 目标
  - 通过 API 暴露系统能力

### Event-driven Integration

- Capability
  - Event Mesh + CPI
- 目标
  - 实时事件架构
- 场景
  - SalesOrder.Created

### User Integration

- Capability
  - 通常使用 SAP Build Work Zone
  - 而不是 Integration Suite
- 目标
  - 用户层整合

### B2B Integration

- Capability
  - Integration Advisor + CPI
- 目标
  - 企业之间系统集成
- 场景
  - Supplier -> EDI -> SAP
