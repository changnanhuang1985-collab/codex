## 5. 推荐目录使用方式

- 架构文档放到 `01_Architecture`
- API 清单和接口设计放到 `02_APIs`
- `iFlow` 设计与清单放到 `03_iFlows`
- Mapping 相关资料放到 `04_Mappings`
- 安全资料放到 `06_Security`
- 排查与监控资料放到 `07_Monitoring`
- 测试资料放到 `08_Testing`
- 运维资料放到 `09_Operations`
- 方法论文档放到 `10_Documentation`
