# Training 目录说明

本目录用于沉淀 `SAP BTP Integration` 培训资料、学习路径、练习任务和官方资源清单。

当前文件：

- `BTP_Integration_Beginner_Bootcamp.md`
  - 入门级教育资料，采用“上手容易、通关不容易”的游戏化设计思路
- `SAP_Integration_Official_Resources.md`
  - SAP 官方学习资源、练习环境、教程、社区与帮助文档清单

建议使用方式：

1. 先看 `BTP_Integration_Beginner_Bootcamp.md`
2. 再按其中每一关对应的资源链接去练习
3. 团队培训时可由 `PMO` 跟节奏，`Senior Consultant` 讲业务意义，`Architect` 讲架构边界
