# AI+咨询+BTP 实验小组

## 1. 定位

本目录用于在 `changnan_codex` 中搭建一个可持续发展的 AI+咨询+BTP 实验小组工作区。目标不是做一次性 AI 试验，而是形成一套可以持续复制、持续增强、持续交付的团队操作系统。

实验小组服务三类场景：

- 咨询方案设计与客户沟通
- SAP BTP / Integration 项目交付与质量提升
- 使用 Codex、VS Code、Agent、自动化脚本持续改善团队效率

## 2. 目标产出

本工作区重点沉淀以下内容：

- 团队使命、愿景、边界与章程
- 组织组件与 AI 角色分工
- 决策机制、会议节奏与治理规则
- 从机会识别到交付复盘的工作流
- 可复用模板与项目启动材料
